터미널이 처음엔 무서워 보이지만, 사실 몇 가지 핵심 개념만 알면 꽤 직관적이에요! 😊

## 터미널이란?

컴퓨터에게 **텍스트로 명령**을 내리는 도구예요. 마우스 클릭 대신 글자로 소통하는 방식이죠.

---

## 기본 명령어 5가지만 알면 시작 가능!

| 명령어 | 의미 | 예시 |
|--------|------|------|
| `pwd` | 지금 어디 있나? (현재 위치) | `/home/user` |
| `ls` | 여기 뭐가 있나? (파일 목록) | `ls` |
| `cd` | 다른 곳으로 이동 | `cd Documents` |
| `mkdir` | 폴더 만들기 | `mkdir 내폴더` |
| `cp / mv` | 파일 복사/이동 | `cp a.txt b.txt` |

---

## 겁먹지 않는 팁

**1. 길을 잃어도 괜찮아요**
```bash
cd ~   # 언제든 홈으로 돌아오기
```

**2. 모르면 물어보세요**
```bash
man ls     # ls 명령어 설명서
ls --help  # 간단한 도움말
```

**3. Tab 키가 자동완성**
```
cd Doc [Tab] → cd Documents/  자동완성!
```

**4. 위쪽 화살표 = 이전 명령어 불러오기**

---

## 터미널을 배우는 가장 좋은 순서

```
현재 위치 확인 → 파일 목록 보기 → 폴더 이동 → 
파일 만들기 → 복사/삭제 → 프로그램 실행
```
