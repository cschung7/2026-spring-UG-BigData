## 핵심 개념

### 💰 국내총생산 (GDP)
**정의**: 한 나라에서 1년간 생산된 모든 최종재화와 서비스의 시장가치

#### 측정 방법
1. **지출접근법**: `GDP = C + I + G + (X - M)`
   - C: 민간소비
   - I: 민간투자
   - G: 정부지출
   - (X-M): 순수출

2. **소득접근법**: `GDP = 임금 + 이윤 + 이자 + 지대 + 간접세`

3. **생산접근법**: `GDP = 각 산업의 부가가치 합계`

#### 실질 vs 명목 GDP
- **명목 GDP**: 당해연도 가격으로 측정
- **실질 GDP**: 기준연도 가격으로 측정
- **공식**: `실질 GDP = (명목 GDP / GDP 디플레이터) × 100`

### 📈 인플레이션과 물가
**정의**: 전반적인 물가수준의 지속적 상승

#### 측정지표
1. **소비자물가지수 (CPI)**
   - `CPI = (현재 장바구니 비용 / 기준연도 장바구니 비용) × 100`
   - `인플레이션율 = ((CPI₁ - CPI₀) / CPI₀) × 100`

2. **GDP 디플레이터**
   - `GDP 디플레이터 = (명목 GDP / 실질 GDP) × 100`

#### 인플레이션의 원인
- **수요견인 인플레이션**: 총수요 증가
- **비용상승 인플레이션**: 생산비용 증가

### 👥 고용과 실업
#### 실업률 계산
```
실업률 = (실업자 수 / 경제활동인구) × 100
경제활동참가율 = (경제활동인구 / 생산가능인구) × 100
```

#### 실업의 유형
1. **마찰적 실업**: 구직과정에서 발생
2. **구조적 실업**: 기술변화나 산업구조 변화
3. **경기적 실업**: 경기침체로 인한 실업
4. **자연실업률**: 마찰적 + 구조적 실업률

## 주요 공식

### 🔄 승수효과
1. **정부지출승수**
   ```
   승수 = 1 / (1 - MPC)
   ```

2. **조세승수**
   ```
   조세승수 = -MPC / (1 - MPC)
   ```

3. **균형예산승수**
   ```
   균형예산승수 = 1
   ```

**용어설명**:
- MPC (한계소비성향): 소득 1단위 증가시 소비 증가분
- MPS (한계저축성향): 소득 1단위 증가시 저축 증가분
- `MPC + MPS = 1`

### 💸 화폐와 이자율
#### 화폐수량설명식
```
M × V = P × Y
```
- M: 화폐공급량
- V: 화폐유통속도
- P: 물가수준
- Y: 실질GDP

#### 피셔효과
```
명목이자율 = 실질이자율 + 기대인플레이션율
```

#### 현재가치와 미래가치
```
현재가치 (PV) = FV / (1 + r)ⁿ
미래가치 (FV) = PV × (1 + r)ⁿ
```

### 📊 경제성장
#### 솔로우 성장모델
```
경제성장률 = 기술진보율 + α × 자본증가율 + (1-α) × 노동증가율
```
- α: 자본의 산출탄력성

## 경제지표 이해

### 📈 주요 경기지표
| 지표 | 의미 | 정상범위 |
|------|------|----------|
| GDP 성장률 | 경제성장 속도 | 2-4% |
| 인플레이션율 | 물가상승률 | 2-3% |
| 실업률 | 고용상황 | 3-5% |
| 기준금리 | 통화정책 기조 | 상황에 따라 |

### 🔄 경기순환
1. **확장**: GDP↑, 실업률↓, 인플레이션↑
2. **정점**: 경제활동 최고조
3. **수축**: GDP↓, 실업률↑, 인플레이션↓
4. **저점**: 경제활동 최저점

### 📉 필립스 곡선
**단기**: 인플레이션과 실업률 간 음의 상관관계
```
π = πᵉ - β(u - uₙ)
```
- π: 인플레이션율
- πᵉ: 기대인플레이션율
- u: 실업률
- uₙ: 자연실업률

## 경제모델

### 📊 IS-LM 모델
#### IS곡선 (재화시장 균형)
```
Y = C + I + G
소득↑ → 이자율↑ (투자감소)
```

#### LM곡선 (화폐시장 균형)
```
M/P = L(Y, r)
소득↑ → 화폐수요↑ → 이자율↑
```

### 🌍 개방경제 모델
#### 무역수지
```
무역수지 = 수출 - 수입
경상수지 = 무역수지 + 소득수지 + 경상이전수지
```

#### 구매력평가설 (PPP)
```
환율 = 국내물가 / 해외물가
```

## 정책도구

### 💰 재정정책
#### 확장적 재정정책
- 정부지출 증가 ↗️
- 세율 인하 ↗️
- **효과**: GDP↑, 인플레이션↑

#### 긴축적 재정정책
- 정부지출 감소 ↘️
- 세율 인상 ↘️
- **효과**: GDP↓, 인플레이션↓

### 🏦 통화정책
#### 확장적 통화정책
- 기준금리 인하 ↘️
- 통화공급량 증가 ↗️
- **효과**: 투자↑, 소비↑, GDP↑

#### 긴축적 통화정책
- 기준금리 인상 ↗️
- 통화공급량 감소 ↘️
- **효과**: 투자↓, 소비↓, GDP↓

## 연습문제

### 🧮 계산 문제

#### 문제 1: GDP 계산
한국의 2024년 경제지표가 다음과 같다:
- 민간소비: 900조원
- 민간투자: 300조원
- 정부지출: 250조원
- 수출: 800조원
- 수입: 750조원

**지출접근법으로 GDP를 계산하시오.**

<details>
<summary>정답 보기</summary>

```
GDP = C + I + G + (X - M)
GDP = 900 + 300 + 250 + (800 - 750)
GDP = 900 + 300 + 250 + 50
GDP = 1,500조원
```
</details>

#### 문제 2: 승수효과
한계소비성향이 0.8일 때, 정부가 100조원의 추가 지출을 한다면 국민소득은 얼마나 증가할까?

<details>
<summary>정답 보기</summary>

```
승수 = 1 / (1 - MPC) = 1 / (1 - 0.8) = 1 / 0.2 = 5

국민소득 증가 = 정부지출 증가 × 승수
국민소득 증가 = 100 × 5 = 500조원
```
</details>

#### 문제 3: 실질GDP
2023년 명목GDP가 2,000조원, GDP 디플레이터가 110이라면 실질GDP는?

<details>
<summary>정답 보기</summary>

```
실질GDP = (명목GDP / GDP 디플레이터) × 100
실질GDP = (2,000 / 110) × 100 = 1,818.2조원
```
</details>

### 💭 개념 문제

#### 문제 4: 인플레이션의 영향
인플레이션이 경제에 미치는 긍정적 영향과 부정적 영향을 각각 3가지씩 설명하시오.

<details>
<summary>정답 보기</summary>

**긍정적 영향:**
1. 명목 부채 부담 감소
2. 경제성장 촉진 (적절한 수준)
3. 디플레이션 방지

**부정적 영향:**
1. 구매력 감소
2. 소득분배 악화
3. 투자 불확실성 증가
</details>
