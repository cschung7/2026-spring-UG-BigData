## 🧮 기본 계산 예제

### 💰 GDP 계산

#### 예제 1: 한국 2023년 GDP 계산
**주어진 데이터:**
- 민간소비(C): 1,042.1 조원
- 민간투자(I): 645.9 조원  
- 정부지출(G): 412.8 조원
- 수출(X): 728.6 조원
- 수입(M): 588.6 조원

**계산과정:**
```
GDP = C + I + G + (X - M)
GDP = 1,042.1 + 645.9 + 412.8 + (728.6 - 588.6)
GDP = 1,042.1 + 645.9 + 412.8 + 140.0
GDP = 2,240.8 조원
```

**해석:** 2023년 한국의 명목 GDP는 약 2,241조원이며, 이는 전년 대비 약 2.7% 증가한 수치입니다.

#### 예제 2: 실질 GDP 계산
**주어진 데이터:**
- 2023년 명목 GDP: 2,240.8 조원
- GDP 디플레이터: 110.8 (2015년=100)

**계산과정:**
```
실질 GDP = (명목 GDP / GDP 디플레이터) × 100
실질 GDP = (2,240.8 / 110.8) × 100
실질 GDP = 2,023.1 조원
```

**해석:** 물가상승을 제거한 2023년 실질 GDP는 2,023조원으로, 실질 경제성장률은 3.1%입니다.

### 📈 인플레이션 계산

#### 예제 3: 한국 2022년 인플레이션율
**주어진 데이터:**
- 2021년 소비자물가지수: 100.0
- 2022년 소비자물가지수: 105.1

**계산과정:**
```
인플레이션율 = ((현재 CPI - 이전 CPI) / 이전 CPI) × 100
인플레이션율 = ((105.1 - 100.0) / 100.0) × 100
인플레이션율 = 5.1%
```

**해석:** 2022년 한국의 인플레이션율은 5.1%로, 한국은행의 목표(2%)를 크게 상회했습니다.

#### 예제 4: 실질임금 변화 계산
**주어진 데이터:**
- 2022년 명목임금 상승률: 3.2%
- 2022년 인플레이션율: 5.1%

**계산과정:**
```
실질임금 변화율 = 명목임금 상승률 - 인플레이션율
실질임금 변화율 = 3.2% - 5.1% = -1.9%
```

**해석:** 2022년 실질임금은 1.9% 감소하여 근로자의 실제 구매력이 악화되었습니다.

### 👥 고용지표 계산

#### 예제 5: 실업률 계산
**주어진 데이터:**
- 경제활동인구: 27,662,000명
- 취업자: 26,804,000명
- 실업자: 858,000명

**계산과정:**
```
실업률 = (실업자 수 / 경제활동인구) × 100
실업률 = (858,000 / 27,662,000) × 100
실업률 = 3.1%
```

**추가 계산:**
```
고용률 = (취업자 수 / 생산가능인구) × 100
경제활동참가율 = (경제활동인구 / 생산가능인구) × 100
```

### 🔄 승수효과 계산

#### 예제 6: 정부지출승수
**주어진 데이터:**
- 한계소비성향(MPC): 0.75
- 정부지출 증가: 10조원

**계산과정:**
```
정부지출승수 = 1 / (1 - MPC)
정부지출승수 = 1 / (1 - 0.75) = 1 / 0.25 = 4

국민소득 증가 = 정부지출 증가 × 승수
국민소득 증가 = 10 × 4 = 40조원
```

#### 예제 7: 조세승수
**주어진 데이터:**
- 한계소비성향(MPC): 0.8
- 세금 증가: 5조원

**계산과정:**
```
조세승수 = -MPC / (1 - MPC)
조세승수 = -0.8 / (1 - 0.8) = -0.8 / 0.2 = -4

국민소득 감소 = 세금 증가 × 조세승수
국민소득 감소 = 5 × (-4) = -20조원
```


**관찰점:**

- 높은 인플레이션과 낮은 실업률이 동시에 나타남

## 🏛️ 정책효과 분석

### 📈 확장적 재정정책 시뮬레이션

#### 시나리오: 경기침체 대응 정책
**정책내용:**
- 정부지출 20조원 증가
- 소득세율 2%p 인하
- 한계소비성향: 0.75

**예상효과 계산:**

1. **정부지출 효과:**
   ```
   정부지출승수 = 1 / (1 - 0.75) = 4
   GDP 증가 = 20 × 4 = 80조원
   ```

2. **세금인하 효과:**
   ```
   세금감소 = 가처분소득 증가분 (약 15조원 가정)
   조세승수 = -0.75 / (1 - 0.75) = -3
   GDP 증가 = 15 × 3 = 45조원
   ```

3. **총 GDP 증가:** 80 + 45 = 125조원 (약 5.6% 증가)

**부작용:**
- 재정적자 확대
- 인플레이션 압력 증가
- 구축효과(Crowding out) 가능성

### 💰 통화정책 효과 분석

#### 시나리오: 기준금리 1%p 인하
**전달경로:**

1. **이자율 효과**
   - 대출이자↓ → 투자↑, 소비↑
   - 예상 투자 증가: 15조원

2. **환율 효과**
   - 자본유출 → 원화절하 → 수출경쟁력 개선
   - 예상 순수출 증가: 8조원

3. **자산가격 효과**
   - 주식, 부동산 가격 상승 → 부의 효과 → 소비 증가
   - 예상 소비 증가: 12조원

**총 효과:** GDP 약 1.5-2.0% 증가 예상

## 📐 경제모델 응용

### IS-LM 모델 분석

#### 사례: 확장적 재정정책 + 긴축적 통화정책

**정책조합:**
- 정부지출 증가 (IS 곡선 우측 이동)
- 기준금리 인상 (LM 곡선 위쪽 이동)

**결과:**
- GDP 변화: 불확실 (정책 강도에 따라 달라짐)
- 이자율: 확실히 상승
- 투자: 감소 (구축효과)
- 정부부문: 확대

**현실 적용:**
이런 정책조합은 재정확장으로 인한 인플레이션을 통화긴축으로 억제하려 할 때 사용됩니다.

### 솔로우 성장모델

#### 한국의 장기성장 전망

**모델 파라미터 (추정치):**
- 저축률(s): 35%
- 감가상각률(δ): 8%
- 인구증가율(n): 0.1%
- 기술진보율(g): 2.5%
- 자본탄력성(α): 0.35

**정상상태 계산:**
```
k* = [s / (δ + n + g)]^(1/(1-α))
k* = [0.35 / (0.08 + 0.001 + 0.025)]^(1/0.65)
k* = [0.35 / 0.106]^1.54 ≈ 8.2
```

**함의:**
- 한국은 이미 정상상태에 근접
- 향후 성장은 주로 기술진보에 의존
- 저출산으로 인한 성장률 둔화 불가피

## 📰 시사문제 연계

### 2024년 주요 경제이슈

#### 1. 반도체 슈퍼사이클과 한국경제

**배경:**
- AI 붐으로 인한 반도체 수요 급증
- 한국의 메모리 반도체 세계 1위

**경제적 영향:**
```
수출 증가 → GDP 성장 기여 → 고용 증가 → 소비 증가
```

**계산 예시:**
반도체 수출이 100조원 증가한다면:
- 직접효과: GDP 100조원 증가
- 승수효과 (승수=2.5 가정): 총 250조원 증가 (약 11% 성장 기여)

#### 2. 부동산 정책과 거시경제

**정책변화:**
- 대출규제 완화
- 공급 확대 정책

**예상 경로:**
```
부동산 투자↑ → 건설업 활성화 → 고용↑ → 소비↑ → GDP↑
```

**리스크:**
- 가계부채 증가
- 부동산 버블 재발 가능성

#### 3. 인구절벽과 경제성장

**현황:**
- 합계출산율: 0.78명 (세계 최저)
- 고령화율: 18.4% (2023년)

**장기 영향:**
```
노동력 감소 → 잠재성장률 하락 → 사회보장비 증가 → 재정부담 증가
```

**대응방안:**
- 생산성 향상 (기술혁신)
- 여성·고령자 경제활동 참가 확대
- 이민정책 개선

## 💡 실전 문제풀이

### 종합문제 1: 정책 mix 분석

**상황:**
한 국가가 다음과 같은 경제상황에 직면했습니다.
- 실업률: 6.5% (자연실업률 4%보다 높음)
- 인플레이션율: 1.0% (목표 2%보다 낮음)
- 재정적자 비율: GDP의 8%

**질문:**
1. 현재 경제상황을 진단하시오.
2. 적절한 정책조합을 제시하시오.
3. 각 정책의 예상효과를 설명하시오.

**답안:**

1. **경제상황 진단:**
   - 경기침체 상황 (높은 실업률)
   - 디플레이션 위험 (낮은 인플레이션)
   - 재정건전성 악화

2. **정책조합:**
   - **1단계**: 확장적 통화정책 우선 (재정건전성 고려)
   - **2단계**: 선별적 재정지출 (고용창출 중심)

3. **예상효과:**
   ```
   통화정책: 이자율↓ → 투자↑ → 고용↑ → 소비↑
   재정정책: 공공투자↑ → 직접 고용 창출 → 승수효과
   ```

### 종합문제 2: 개방경제 분석

**상황:**
한국의 주요 교역국인 미국이 기준금리를 크게 인상했습니다.

**파급경로 분석:**

1. **단기효과:**
   ```
   미국 금리↑ → 달러 강세 → 원화 절하 → 수입물가↑ → 인플레이션↑
   ```

2. **중기효과:**
   ```
   원화 절하 → 수출경쟁력↑ → 순수출↑ → GDP↑
   ```

3. **정책대응:**
   - 한국도 금리 인상 (자본유출 방지)
   - 환율 안정화 조치
   - 수입 인플레이션 모니터링
