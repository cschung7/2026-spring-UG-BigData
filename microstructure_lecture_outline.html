Part II. First Generation: Price-Only Proxies

가격 또는 OHLC만으로 spread와 volatility를 추정하는 고전적 방법들
7

Tick Rule Intuition

trade signing의 가장 단순한 방식

  • up-tick → buyer initiated
  • down-tick → seller initiated
  • quote가 없을 때 signed flow의 출발점
b_t = 1 if Δp_t > 0 b_t = -1 if Δp_t < 0 b_t = b_(t-1) if Δp_t = 0
8

Tick Rule Applications

단순하지만 surprisingly useful한 이유를 설명

  • Kalman filter expectation of trade sign
  • structural breaks
  • entropy, run tests, fractional differentiation
9

Roll Model: Bid-Ask Bounce

short-run negative autocovariance를 spread 추정에 연결

  • mid-price process
  • transaction price = mid + trade sign × half-spread
  • trade bounce의 핵심 직관
m_t = (p_t^b + p_t^a) / 2 m_t = m_(t-1) + u_t p_t = m_t + b_t c
10

Roll Derivation and Recovery

return covariance로부터 half-spread를 복원

  • E(Δp_t Δp_(t-1)) = -c²
  • c from serial covariance
  • true price innovation variance 추정
Δp_t = u_t + c(b_t - b_(t-1)) E(Δp_t²) = σ_u² + 2c² E(Δp_t Δp_(t-1)) = -c²
11

Extended Roll

inventory cost와 asymmetric information cost를 분리하는 관점

  • public news shock + private information component
  • spread decomposition
  • identification issue 소개
p_t = m_t + c b_t m_t = m_(t-1) + ff_t ff_t = λ b_t + v_t p_t = m_(t-1) + (λ + c)b_t + v_t
12

Parkinson High-Low Volatility

OHLC 범위에서 latent volatility를 추정

  • intraday range as volatility information
  • low-frequency datasets에 유용
  • spread/jump 오염 가능성도 함께 설명
E[(1/T) Σ (log(H_t/L_t))²] = k · σ_HL
13

Corwin–Schultz Spread Proxy

two-bar high-low 구조를 활용한 bid-ask spread proxy

  • book data가 없을 때 강력
  • bond / opaque markets에도 적용 가능
  • H/L가 volatility + spread를 함께 반영
S_t = 2(e^{α_t}-1)/(1+e^{α_t}) α_t = f(β_t, γ_t)
14

  • better OHLC spread estimators

Part III. Second Generation: Strategic Trade and Market Impact

가격만이 아니라 signed volume과 order flow imbalance가 가격을 어떻게 움직이는지 설명
15

Why Move Beyond Price?

첫 세대는 friction을 가격에서 추정했고, 두 번째 세대는 flow가 price를 움직이는 메커니즘을 봅니다.

  • order flow imbalance
  • market depth
  • adverse selection and impact
16

Kyle Model Setup

informed trader, noise trader, market maker의 삼자 구조

  • market maker는 total flow만 관찰
  • price responds linearly to order flow
  • λ is inverse liquidity
y = x + u p = λy + μ x = α + βv
17

Kyle Optimization

informed trader의 최적 주문과 λ의 의미를 도출

  • mispricing exploitation
  • noise trade concealment
  • lower λ = deeper market
π = (v - p)x x = (v - μ - λu)/(2λ) α = -μ/(2λ), β = 1/(2λ)
18

Kyle Empirical Regression

실증에서는 signed volume 회귀로 연결

  • Δp_t on b_t V_t
  • execution fragility indicator
  • small cap / event windows에 직관적
Δp_t = λ(b_t V_t) + ε_t
19

Amihud Illiquidity

daily 데이터에서 가장 실용적인 price impact proxy 중 하나

  • absolute return per dollar volume
  • cross-market screening에 적합
  • intraday data가 없을 때 강력
|Δ log p_t^c| = λ Σ_{t∈B_τ} p_t v_t + ε_t
20

Hasbrouck Lambda

intraday effective trading cost approximation

  • 5-minute bars recommendation
  • signed sqrt-dollar-volume impact
  • execution desk use case에 적합
log(p_{i,τ}) - log(p_{i-1,τ}) = λ_i Σ (b_{i,t} √(p_{i,t} V_{i,t})) + ε_{i,t}
21

Kyle vs Amihud vs Hasbrouck

“어떤 λ를 쓸 것인가?”를 과업 기준으로 정리

  • theory richness
  • data availability
  • execution vs portfolio research
22

2025 Update: ML + Liquidity Estimation

theory-based liquidity estimators와 ML의 결합

  • classical features still matter
  • daily data + nonlinear models
  • microstructure theory is not replaced, but upgraded

Part IV. Third Generation: Sequential Informed Trading

거래가 순차적으로 도착하는 환경에서 정보거래와 toxicity를 추정하는 접근
23

Sequential Model Intuition

market maker가 거래 흐름을 보며 동적으로 가격을 갱신하는 상황

  • not just large trades
  • informed vs uninformed arrivals
  • adverse selection made explicit
24

PIN Model Tree Logic

news arrival probability와 trader arrival rates로 informed trading probability를 구성

  • α: news event probability
  • δ: good/bad news split
  • μ, ε: informed / uninformed arrival rates
25

PIN Math and Quote Setting

market maker가 손실을 피하기 위해 bid/ask를 어떻게 설정하는지 설명

  • expected terminal value
  • bid / ask adjustment
  • MLE estimation idea
E[S_1] = (1-α)S_0 + α(δS_B + (1-δ)S_G) PIN = αμ / (αμ + 2ε)
26

PIN Interpretation

“현재 거래 중 어느 정도가 정보를 가진 거래일까?”

  • takeover rumor case
  • one-sided order flow
  • market maker caution
27

VPIN and Volume Time

calendar time 대신 volume clock으로 toxicity를 추적

  • equal-volume buckets
  • flow imbalance as liquidity-stress signal
  • flash-crash narrative connection
VPIN_t = Σ|V_τ^B - V_τ^S| / Σ(V_τ^B + V_τ^S)
28

PIN vs VPIN

structural daily metric과 high-frequency rolling metric을 비교

  • PIN: slower, model-based
  • VPIN: faster, imbalance-based
  • task-dependent choice
29

Modern Caution on VPIN

최근 연구는 realized imbalance보다 latent book shape가 더 informative할 수 있음을 시사

  • toxicity ≠ full story
  • latent supply-demand slope matters
  • volatility prediction task에서 비교 필요

Extended Roll Bid / Ask

Extended Roll Moments
